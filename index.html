計算工坊 (ArkFactory) 隱私權政策

最後更新日期：2026年6月14日

計算工坊（以下簡稱「本 App」）非常重視您的隱私。本政策說明我們如何處理您的資訊：

1. 資訊收集與使用（網路與匯率）

本 App 的核心計算與歷史紀錄皆儲存於您的個人設備中。為了提供精準的國際工程單位與匯率換算，本 App 會透過您的 IP 位址進行本地網絡判斷以自動切換對應國家匯率，我們不會在伺服器端端記錄或追蹤您的具體地理定位。

2. 廣告與營利服務

本 App 未來可能包含第三方廣告服務（如 Google AdMob）以支援營運。這些廣告 SDK 可能會收集匿名裝置識別碼（如 IDFA 或 GAID）以提供基礎廣告投放與流量數據統計。相關規範皆符合第三方官方政策。

3. 資料安全性與註冊

本 App 屬於工具類軟體，無需註冊會員即可使用，我們不會主動收集任何使用者的個人姓名、電話、電子郵件等敏感個資。

4. 聯絡我們

若對本隱私政策有任何疑問，請透過 GitHub 倉庫提交 Issue 聯繫開發者。
