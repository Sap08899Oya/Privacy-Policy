計算工坊 隱私權政策

最後更新日期：2026年6月14日

計算工坊（以下簡稱「本 App」）非常重視您的隱私。本政策說明我們如何處理您的資訊。

1. 資訊下載與使用

本 App 的核心計算與紀錄皆儲存於您的個人設備本機中，不會上傳至任何外部伺服器。為提供即時匯率換算功能，本 App 僅會單向連接以下公開 API 以取得最新的匯率大盤資料：

  • open.er-api.com：取得國際即時匯率基準資料。

此連線僅用於下載公共數據，本 App 絕不會收集、傳輸或記錄您的個人身份識別資訊、網路 IP 位址或地理位置。

2. 本地資料儲存

所有計算紀錄（包含貸款還款進度、ETF 持倉資料等）僅儲存於您的設備本機，本 App 不會將上述資料上傳至任何伺服器或第三方平台。

3. 廣告與營利服務

本 App 目前為無廣告版本。未來若為支援營運而導入第三方廣告服務（如 Google AdMob），將同步更新本政策並於 App 內提示使用者。屆時廣告 SDK 可能依據官方政策收集匿名裝置識別碼（如 GAID）。

4. 無需註冊・不收集個資

本 App 屬於工具類軟體，無需註冊會員即可使用。我們不會主動收集任何使用者的個人姓名、電話、電子郵件等敏感個資。
