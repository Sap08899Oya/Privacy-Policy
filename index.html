計算工坊 隱私權政策

最後更新日期：2026年6月14日

計算工坊（以下簡稱「本 App」）非常重視您的隱私。本政策說明我們如何處理您的資訊。

1. 資訊下載與使用

本 App 的核心計算與紀錄皆儲存於您的個人設備本機中，不會上傳至任何外部伺服器。為提供即時匯率換算功能，本 App 僅會單向連接以下公開 API 以取得最新的匯率大盤資料：

  • open.er-api.com：取得國際即時匯率基準資料。

此連線僅用於下載公共數據，本 App 絕不會收集、傳輸或記錄您的個人身份識別資訊、網路 IP 位址或地理位置。

2. 本地資料儲存

為提供持續追蹤與跨次使用的體驗，本 App 使用 Android 系統內建的本地儲存機制（SharedPreferences）將以下資料儲存於您的設備本機：

  • 能源計算車間：每月用電量紀錄（1月至12月，共12筆數值）。
  • ETF 息收：持倉資料，包含 ETF 代號、持有股數、每股配息、買入均價、目前現價及配息頻率。
  • 貸款攤還：還款追蹤狀態，包含原始本金、剩餘本金、已還期數、合約總期數及目前利率。

上述所有資料：

  • 僅儲存於您的個人設備，不會上傳至任何伺服器或第三方平台。
  • 您可隨時於 Android 系統設定 → 應用程式 → 計算工坊 → 儲存空間 → 清除資料，永久刪除所有本地紀錄。
  • 解除安裝本 App 時，所有本地儲存資料將一併自動清除。

3. 廣告與營利服務

本 App 目前為無廣告版本。未來若為支援營運而導入第三方廣告服務（如 Google AdMob），將同步更新本政策並於 App 內提示使用者。屆時廣告 SDK 可能依據官方政策收集匿名裝置識別碼（如 GAID）。

4. 無需註冊・不收集個資

本 App 屬於工具類軟體，無需註冊會員即可使用。我們不會主動收集任何使用者的個人姓名、電話、電子郵件等敏感個資。

若對本隱私政策有任何疑問，請透過電子郵件與我們聯繫：spp.pans@gmail.com
